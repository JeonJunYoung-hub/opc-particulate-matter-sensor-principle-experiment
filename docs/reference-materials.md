Large vendor PDFs and local lab documents are not committed to this repository. For reproducibility, look up the current public vendor documents for:

- Hamamatsu C9118 driver circuit
- Hamamatsu S11865-64G photodiode array
- Plantower PMS5003 particulate matter sensor
- Sensirion SPS30 particulate matter sensor
- Sharp GP2Y1010AU optical dust sensor
- TSI aerodynamic particle sizer documentation, if using it as a lab reference

Keep downloaded PDFs, slide decks, and local experiment notes outside Git.
